| Milestone | Completion Definition | Evidence |
| --- | --- | --- |
| M17-1 文档冻结 | PRD、目标架构、开发验收计划、验收门槛、gap drawio、P17 contract 全部描述同一目标 | 文档审计、drawio XML 校验、审计路径清单 |

| M17-7 性能与总验收 | 真实数据组合下缩放、筛选、详情打开可用；命令和 PRD 复检通过 | build/test/e2e/verify:scanlist、性能记录、最终验收摘要 |

## P17 出门条件

- 完整用户历程可在移动端跑通：地图 -> 待确认 -> 详情 -> 标签编辑 -> 挪动图钉 -> 打开地图 -> 分享图。
- 待确认地点必须可处理：确认候选、跳过、手动挪动至少三种动作可用。
- 详情页在目标尺寸下不出现横向裁切、不可滚动、地图跟随滚动空白或底部栏遮挡。
- 首页筛选栏在桌面、平板、移动端所有可见控件完整可见。
- 图钉视觉系统稳定区分已核验收藏、待确认收藏、选中态、扫街榜、参考图层和高密度状态。
- 分享图导出反映当前筛选结果，不导出空白或错集。
- 真实数据验收覆盖扫街榜 50 条、钉图易参考层、个人收藏和待确认队列。
- `npm run build`、`npm test`、`npx playwright test`、`npm run verify:scanlist` 全部通过。
- P17 final acceptance report 或 P17 验收摘要必须创建后才能宣布出门。

Historical milestones are archived under `docs/history`; this active roadmap tracks P17.
