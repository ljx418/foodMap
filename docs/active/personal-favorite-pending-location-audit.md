# 个人收藏待确认地点全量位置审计

## 结论

当前个人收藏共 32 个上图点，其中 1 个为精确坐标，31 个为待确认位置。待确认位置全部不得视为真实门店坐标，只能作为用户记忆线索和后续校准入口。

本轮修正把所有待确认点统一标记为 `位置待确认`；其中名称泛化、公开来源不稳定、仅有商圈/候选描述或已被用户指出错误的地点增加 `位置高风险`。高风险点必须通过用户补充截图/门牌、候选确认或手动挪动后，才能解除风险。

## 后续验收门槛

- `位置待确认` 不得解锁精确导航。
- `位置高风险` 必须在详情页显示风险提示。
- 坐标落入中心城区疑似水域时必须触发风险提示。
- 用户手动挪动或候选确认后，才能移除 `位置高风险` 和 `位置待确认`。
- 任何自动迁移不得覆盖用户已经手动挪动过的坐标。
