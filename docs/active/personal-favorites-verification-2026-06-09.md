## 验收要求

- Clean profile 启动后，地图不得自动出现个人收藏图钉。
- 用户点击 `导入我的收藏清单` 后，地图应出现 `我的收藏` 图层和 `32` 个个人收藏图钉。
- 其中 `1` 个为精确坐标图钉，`31` 个必须带有 `待校准` 和 `近似坐标` 标签。
- 待校准图钉详情不得展示可点击的精确导航主入口，只允许复制地址/坐标，避免把近似点误导为精确导航点。
- 首页标签筛选应能过滤这些个人收藏，例如 `吃过`、`火锅`、`湖北菜`、`百分制93`。
- 待确认条目不得出现在正式图钉中。
- 详情页备注必须显示原始记录、百分制评分、核验置信度和核验证据摘要。

### 后续复核规则

- 只要用户指出某个已导入个人收藏点位错误，应优先检查是否为“连锁/同名/道路泛点”导致的误定位。
- 未取得门牌或可靠 POI 坐标时，只能作为待校准图钉展示。
- 如果只能确认城市或大商圈，详情必须提示“待校准/不作为精确导航依据”。
- 已经导入过的 `personal-favorite:` 近似生成图钉会被迁移为待校准图钉；clean profile 仍不得自动写入个人图钉。
